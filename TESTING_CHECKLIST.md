## 🐛 버그 리포팅

### 버그 발견 시 기록 항목
1. **재현 단계**: 상세한 단계별 설명
2. **예상 결과**: 올바른 동작 설명  
3. **실제 결과**: 실제 발생한 문제
4. **환경 정보**: 기기, OS 버전, 앱 버전
5. **스크린샷/동영상**: 가능한 경우 첨부
6. **빈도**: 항상/가끔/한번만 발생

### 우선순위 분류
- **P0 (Critical)**: 앱 크래시, 핵심 기능 불가
- **P1 (High)**: 주요 기능 오동작
- **P2 (Medium)**: 사소한 UI 이슈
- **P3 (Low)**: 개선 제안
